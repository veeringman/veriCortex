# VeriCortex SDK Suite

This repository contains SDKs enabling integration with the VeriCortex inference proof system.

## Structure
- **core/** — Rust & TypeScript SDKs
- **ai/** — Python SDK for model proofs
- **blockchain/** — Solidity smart contracts
- **mobile/** — Android and iOS SDKs
- **unity/** — Game development support

Each SDK provides standardized APIs to interact with on-chain and off-chain proof systems.
