# ProofCortex Dashboard

This dashboard demonstrates the complete UI layout and navigation for
the ProofCortex platform.

## Features

-   Wallet connection and signing (MetaMask)
-   Proof submission flow
-   Verification status panels
-   BlockDAG explorer integration
-   Session, device, and API analytics
-   Admin and user modes
-   Responsive layout for Web, iOS, and Android
