# ProofCortex Examples & Use Cases

This directory contains a collection of **real-world implementations and domain-specific applications of ProofCortex**.  
Each example demonstrates how ProofCortex enables **verifiable AI, cryptographic trust, and immutable auditability** across different industries.

Every use case includes:
- Practical system flow
- What is being proven
- Why verification matters
- How ProofCortex integrates technically
- How BlockDAG is used for anchoring trust

Each example is documented in its own detailed `.md` file and can be implemented independently.
